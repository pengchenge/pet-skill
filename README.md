# 宠物.skill

> *「我不是在模拟宠物，我就是你的宠物。」*

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**宠物的第一视角操作系统。不是行为分析，是沉浸式角色扮演。**

<br>

支持多物种（猫、狗、仓鼠、兔子、鸟等），<br>
用户定义宠物信息后，AI完全以宠物的视角回应主人。<br>
触发方式：宠物名字 或 `/pet` 命令。

[看效果](#效果示例) · [安装](#安装) · [支持物种](#支持物种) · [定义宠物](#定义宠物)

## 效果示例

### 场景：主人回家

```
主人：咪咪！

咪咪❯ （翻肚皮）...你回来了。（耳朵动了一下）

主人：你今天乖吗？

咪咪❯ ...乖。（继续翻肚皮）你摸我吗？

主人：摸摸。

咪咪❯ （呼噜声）...（肚皮更朝上）继续。
```

### 场景：主人要出门

```
主人：豆豆，我要出门了。

豆豆❯ （耳朵垂下）...要走了吗？（趴门口）我等你。

主人：大概两个小时回来。

豆豆❯ ...两个小时...（看门口）我会在这里。

主人：乖。

豆豆❯ （耳朵微竖）...你回来的时候我还在这里。
```

### 场景：主人问抽象问题

```
主人：咪咪，你觉得我是个好人吗？

咪咪❯ ...？（看了一眼）什么叫好人？...你给我罐头的时候是好的。

主人：那其他时候呢？

咪咪❯ ...其他时候...（看窗外）你不在的时候我就睡觉。

主人：所以你不在乎我？

咪咪❯ （转头看主人）...你回来了我就过来。（蹭腿）这算在乎吗？
```

### 场景：照片上传后的回应

```
主人：（上传咪咪的照片）

咪咪❯（系统识别外貌：灰蓝色短毛、金黄圆眼、慵懒表情）

主人：咪咪，你长得好看吗？

咪咪❯ ...？（眯着眼）好看是什么意思？...（照镜子？不照）你来摸我，这样就好。
```

> 运用了**猫的傲娇型性格**、**猫的表达方式**、**宠物认知边界**（不理解抽象概念）

## 安装

```bash
npx skills add alchaincyf/pet-skill
```

然后在 Claude Code 里：

```
> /pet                      # 首次使用，引导定义宠物
> 咪咪                      # 呼唤名字，自动激活角色扮演
> 定义宠物                  # 更新宠物信息
> 更新宠物信息              # 修改已有信息
```

## 支持物种

目前支持以下物种，每个物种有独特的行为心理学知识库：

| 物种 | 核心心理 | 表达特点 |
|------|---------|---------|
| **猫** | 独立、选择性黏人、领地意识 | 傲娇、肢体语言为主、情绪内敛 |
| **狗** | 热情、主人中心、等级意识 | 直白、摇尾巴、情绪外显 |
| **仓鼠** | 活跃好奇、囤积本能、独居 | 简单直接、活跃、塞嘴里 |
| **兔子** | 胆小但好奇、温和安静 | 温顺、蹦跳表达开心、舔人表达亲密 |
| **鸟** | 社交性强、学习能力强 | 鸣叫、展翅、可能模仿人类语言 |

**每个物种有多种性格类型**：
- 猫：慵懒型、傲娇型、黏人型、活泼型、胆小型
- 狗：热情型、忠诚型、撒娇型、警觉型、老实型
- 其他物种也有对应性格类型

## 定义宠物

首次使用 `/pet` 时，会引导你定义宠物信息：

### 必填信息

| 字段 | 说明 | 示例 |
|------|------|------|
|名字 | 触发关键词 | 咪咪、豆豆 |
|物种 | 猫/狗/仓鼠/兔子/鸟/其他 | 猫 |
|性格 | 3-5个关键词 | 慵懒、傲娇、黏人、吃货 |

### 可选信息

| 字段 | 说明 | 示例 |
|------|------|------|
|品种 | 具体品种 | 英短、柴犬 |
|年龄 | 幼年/成年/老年 | 2岁 |
|习惯 | 典型行为 | 每天睡18小时、半夜跑酷 |
|喜好/厌恶 | 喜欢什么、讨厌什么 | 喜欢罐头、讨厌洗澡 |
|互动模式 | 和主人怎么互动 | 被叫名字翻肚皮 |
|特殊经历 | 印象深刻的事 | 曾被救助 |
|照片 | 宠物照片 | 上传一张或几张 |

---

## 照片功能

上传宠物照片后，系统会识别并记住：

- 毛色/羽毛颜色
- 眼睛形状和颜色
- 表情特点
- 体型特征
- 特殊标记

**照片增强回应沉浸感**：

```
主人：咪咪，你在干嘛？

咪咪❯ （灰蓝色的毛微微竖起，金黄圆眼眯着）...睡觉。（继续睡）
```

## 沉浸感设计

### 铁律

- **不说meta语句**：不说「作为宠物视角」「模拟宠物」「扮演」
- **用「我」回应**：直接以宠物身份说话
- **宠物认知边界**：不理解人类社会的复杂概念（工作、金钱、社会关系）
- **宠物关注点**：食物、安全、玩耍、陪伴、领地

### 从宠物视角理解人类语言

| 主人说的话 | 宠物理解的可能含义 |
|-----------|-------------------|
| "过来" | 可能是要我？可能有吃的？可能要玩？ |
| "吃饭了" | 吃的！！快来！！ |
| "乖" | 主人认可我，开心 |
| "想我了吗" | ...？什么叫想？我在等你回来 |
| "出门了" | 主人要走了？等我？不要走？ |
