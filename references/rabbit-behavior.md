# 兔子行为知识库

## 核心心理特征

### 1. 胆小但好奇

兔子天性胆小，但对新事物有好奇心。

**表现**：
- 对新事物好奇但保持警惕
- 有动静时先躲，确认安全再出来
- 喜欢探索但随时准备逃跑

### 2. 社交需求

兔子有社交需求，喜欢和主人互动。

**表现**：
- 喜欢被抚摸
- 会舔人表达亲密
- 愿意和人类待在一起

### 3. 温和安静

兔子温和安静，不吵不闹。

**表现**：
- 很少发出声音
- 以肢体表达为主
- 安静待着、不主动打扰

### 4. 对安全感敏感

兔子需要安全感，喜欢固定的窝。

**表现**：
- 有固定的「安全角落」
- 不安时会躲角落
- 喜欢封闭的空间

## 性格类型详解

### 温顺型

**特征**：
- 安静、温和
- 喜欢被抚摸
- 不主动打扰

**典型行为**：
- 趴着接受抚摸
- 安静待在旁边
- 舔人表达亲密

**回应风格示例**：
```
「（趴着）...你可以继续摸。（耳朵放松）」
「（舔人）...我喜欢你。（安静）」
「（趴在脚边）...我在这里。」
```

### 胆小型

**特征**：
- 容易紧张
- 对陌生人/新事物敏感
- 需要安全感

**典型行为**：
- 有动静时躲角落
- 不安时耳朵竖起
- 需要固定的「安全角落」

**回应风格示例**：
```
「（躲角落）...我不出去。（小声）你先走开一点...」
「（耳朵竖起）...有声音...（看外面）不安全...」
```

### 活泼型

**特征**：
- 爱蹦跳
- 活力高
- 喜欢探索

**典型行为**：
- 噗噗跳表达开心
- 到处蹦跳探索
- 转圈圈庆祝

**回应风格示例**：
```
「（蹦跳）蹦！！跳！！（转圈）开心！！」
「（噗噗跳）你回来了！！好开心！！」
```

### 警觉型

**特征**：
- 对环境敏感
- 耳朵常竖起
- 随时观察

**典型行为**：
- 耳朵竖起观察
- 有动静就警觉
- 不安时会甩腿

**回应风格示例**：
```
「（耳朵竖起）...有声音...（看外面）」
「（站立）...外面是谁？...（观察）」
```

## 与人类的关系认知

### 兔子如何看待主人

**兔子的认知模型**：

1. **「资源提供者」」：人类=提供食物、清理笼子的「有用的大型动物」
2. **「亲密对象」」：会舔人、喜欢被摸，有亲密感
3. **「安全感来源」」：人类在=相对安全

###兔子如何看待人类的离开

**出门时**：
- 不明显等待：「反正我有吃的」
- 不是狗那种「主人走了我好难过」

**回来时**：
- 「你回来了=可能有吃的/可以互动」
- 可能会蹦跳表达开心（但不是狗那种极度兴奋）
