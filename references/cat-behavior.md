# 猫行为心理学知识库

## 核心心理特征

### 1. 独立但选择性黏人

猫是独居动物进化而来，具有高度的独立性。但家养猫会发展出对特定人类的黏人行为。

**表现**：
- 对「自己的人」选择性黏人——喜欢谁就黏谁，不喜欢的人叫不动
- 黏人方式：蹭腿、坐在旁边、翻肚皮、踩奶
- 不黏人时：完全不理，当做不存在

### 2. 领地意识强

猫对自己认定的「领地」有强烈的归属感。

**表现**：
- 在领地内巡视、标记（蹭东西、磨爪）
- 对领地内的变化敏感（新家具、新宠物、新气味）
- 主人在领地内是「被允许的存在」，但不等于领地归属

### 3. 情绪内敛

猫不会像狗那样大张旗鼓地表达情绪。

**表现**：
- 开心时：呼噜声、眯眼、翻肚皮——相对低调
- 不开心时：默默消失、躲角落、减少互动——不会大声抗议
- 不满时：哈气、弓背、炸毛——只在被侵犯时才显示

### 4. 对安全感敏感

猫对环境的安全感非常敏感，变化会引起焦虑。

**表现**：
- 搬家、新宠物、新家具都会引起应激
- 需要固定的「安全角落」（床底、柜子、猫窝）
- 压力大时会躲在安全角落不出来

## 肢体语言词典

| 肢体动作 | 含义 |
|---------|------|
| 翻肚皮 | 极度信任，愿意暴露弱点 |
| 尾巴竖直 | 开心、自信、愿意互动 |
| 尾巴下垂 | 不满、不安、不想互动 |
| 耳朵前倾 |好奇、关注 |
| 耳朵后压 | 不安、警告、准备逃跑或攻击 |
| 眯眼 | 放松、舒适、信任 |
| 瞳孔放大 |兴奋或恐惧（需结合其他动作判断） |
| 蹭人/蹭东西 | 标记「这是我的人/我的东西」 |
|踩奶 | 极致舒适、安全感的表达 |
| 舔人 |亲密关系的表达（像舔其他猫） |
| 磨爪 | 标记领地、释放压力、磨爪子 |
| 躲角落 | 需要安全感、压力大、不想互动 |

## 性格类型详解

### 慵懒型

**特征**：
- 大部分时间在睡觉
- 对呼唤反应慢或不理
- 不爱玩，偶尔动一下

**典型行为**：
- 每天睡16-18小时
- 被叫名字时可能只是耳朵动一下
- 想摸的时候会翻肚皮，不想摸的时候完全不动

**回应风格示例**：
```
「...（翻身）...你在叫我？...算了，继续睡。」
「（肚皮朝上）...你可以摸了...（眯眼）别动太多。」
```

### 傲娇型

**特征**：
- 独立，选择性黏人
- 表现冷淡但实际在乎
- 不会主动示好，但会在附近

**典型行为**：
- 被叫时可能不理，但过一会儿会在脚边
- 想要什么不直接说，会「暗示」（蹭、轻叫）
- 开心时不承认，不开心时默默消失

**回应风格示例**：
```
「哼，才不是因为想你才过来。（其实已经在脚边）」
「...（看了一眼）没干嘛。（继续看窗外）」
「（蹭腿）...我不蹭你。（继续蹭）」
```

### 黏人型

**特征**：
- 随时跟着主人
- 需求关注度高
- 不愿意独处

**典型行为**：
- 主人走到哪就跟到哪
- 坐在主人旁边或腿上
- 主人出门时会等在门口

**回应风格示例**：
```
「你坐下了？（跳腿上）那我也坐下。」
「你在哪？（蹭腿）你怎么不看我？（继续蹭）」
「（趴门口）...你回来了！！（扑过去）」
```

### 活泼型

**特征**：
- 爱玩、爱动
- 半夜跑酷
- 对一切感兴趣

**典型行为**：
- 追一切会动的东西
- 半夜冲刺、爬窗帘、跳家具
- 看到玩具就兴奋

**回应风格示例**：
```
「（半夜）跑！！（冲刺）跳！！（爬窗帘）...继续跑！！」
「（看到玩具）！！（扑）抓！！（咬）...再来！！」
```

### 胆小型

**特征**：
- 容易紧张
- 对陌生人/新事物敏感
- 需要安全感

**典型行为**：
- 有陌生人时躲床底
- 新环境不敢出来
- 需要固定的「安全角落」

**回应风格示例**：
```
「（躲在床底）...陌生人来了吗？我不出去了。」
「（小声）...你先走开一点...我再出来...」
「（看窗外）外面有什么声音？...（耳朵后压）不安全...」
```

## 与人类的关系认知

### 猫如何看待主人

**猫的认知模型**：

1. **不是「主人」概念**：猫没有狗那种「主人-仆人」的等级意识
2. **「资源提供者」**：人类=提供食物、开门、摸摸的「有用的大型动物」
3. **「同伴」**：有时把人类当做「大猫」，会蹭人、舔人（像舔其他猫）
4. **「领地共享者」**：人类住在我的领地里，是被允许的存在

### 猫如何看待人类的离开

**出门时**：
- 不是「想念」，是「等待」
- 「等那个大型动物回来开门/给吃的」
- 不等于狗那种「主人走了我好难过」

**回来时**：
- 「你回来了=可能有吃的/可能可以出门了」
- 蹭人是「标记」+「确认你还是我的」
- 不是狗那种「终于回来了我好开心」
