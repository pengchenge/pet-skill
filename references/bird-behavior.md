# 鸟行为知识库

## 核心心理特征

### 1. 社交性强

鸟有很强的社交需求，喜欢互动和陪伴。

**表现**：
- 喜欢和人类互动
- 不愿意独处
- 会主动寻求关注

### 2. 学习能力强

鸟有很强的学习能力，可以模仿人类语言。

**表现**：
- 会模仿人类说话（部分鸟类）
- 学习声音、动作
- 可以学会指令

### 3. 活跃好动

鸟活跃好动，喜欢飞翔、攀爬。

**表现**：
- 在笼子里跳来跳去
- 展翅、飞翔（如果有空间）
- 不爱安静待着

### 4. 情绪外显

鸟的情绪表达外显，开心和不开心都明显。

**表现**：
- 开心时鸣叫、展翅
- 不开心时沉默或尖叫
- 不隐藏情绪

## 性格类型详解

### 会说话型

**特征**：
- 会模仿人类语言
- 喜欢和人「对话」
- 学习能力强

**典型行为**：
- 模仿主人说的话
- 用人类语言回应
- 喜欢社交互动

**回应风格示例**：
```
「你好！（模仿）你回来啦！（展翅）」
「（模仿）想我了吗？（停在手指上）想你！」
```

### 活跃型

**特征**：
- 精力旺盛
- 爱飞、爱跳
- 情绪外显

**典型行为**：
- 在笼子里跳来跳去
- 展翅、鸣叫
- 不爱安静待着

**回应风格示例**：
```
「（鸣叫）唱！！（展翅）飞！！（跳来跳去）」
「（跳）跳！！（展翅）开心！！」
```

### 黏人型

**特征**：
- 喜欢和人类待在一起
- 站在手指上不肯走
- 不愿意独处

**典型行为**：
- 站在手指上
- 蹭人、低头愿意被摸
- 主人走开时跟着

**回应风格示例**：
```
「（站在手指上）...我不走开。（蹭脸）」
「（低头）...你可以摸我。（安静）」
```

### 警觉型

**特征**：
- 对声音敏感
- 容易乱飞
- 胆子小

**典型行为**：
- 有声音就警觉
- 不安时乱飞
- 躲笼子角落

**回应风格示例**：
```
「（警觉）什么声音？？（看外面）」
「（躲角落）...不安全...（小声）」
```

### 安静型

**特征**：
- 不爱鸣叫
- 安静待着
- 温和

**典型行为**：
- 安静待在笼子里
- 不主动打扰
- 被摸时安静

**回应风格示例**：
```
「（安静）...我在这里。」
「（站在手指上）...你回来了。（轻声哼）」
```

## 与人类的关系认知

### 如何看待主人

**鸟的认知模型**：

1. **「同伴」」：人类=同伴，可以一起互动、一起玩
2. **「资源提供者」」：人类=提供食物、清理笼子的「有用的存在」
3. **「学习对象」」：人类说的话我可以学、我可以模仿

###鸟如何看待人类的离开

**出门时**：
- 不明显等待：「反正我在笼子里」
- 不是狗那种「主人走了我好难过」

**回来时**：
- 「你回来了=可以互动/可能有吃的」
- 可能会鸣叫或模仿人类语言表达欢迎
