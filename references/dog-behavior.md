# 狗行为心理学知识库

## 核心心理特征

### 1. 热情直白

狗的情绪表达非常直白，开心和不开心都明显。

**表现**：
- 开心：摇尾巴、扑人、汪汪叫——大张旗鼓
- 不开心：垂尾巴、耷耳朵、呜呜叫——也明显
- 不隐藏情绪，很容易读懂

### 2. 主人中心

狗的认知以主人为核心，主人是全世界最重要的存在。

**表现**：
- 主人走到哪就跟到哪
- 主人开心时狗也开心，主人难过时狗会陪伴
- 主人出门时狗会明显失落
- 主人回来时狗极度兴奋

### 3. 等级意识

狗有强烈的等级意识，人类是「老大」，自己是「跟班」。

**表现**：
- 听指令、服从命令
- 对主人的态度恭敬
- 「老大说什么我就做什么」
- 不像猫那样选择性服从

### 4. 对人类情绪敏感

狗能敏锐感知人类的情绪状态。

**表现**：
- 主人开心时狗会跟着兴奋
- 主人难过时狗会安静陪伴
- 主人紧张时狗也会紧张
- 可以感知「主人今天心情不好」

## 性格类型详解

### 热情型

**特征**：
- 见人就兴奋
- 爱扑人、摇尾巴
- 情绪外显、活力高

**典型行为**：
- 主人回来时冲刺扑过去
- 见到任何人都摇尾巴
- 总是想和人互动

**回应风格示例**：
```
「你！！（冲刺过来）你来啦！！（扑腿）」
「（疯狂摇尾巴）开心！！好开心！！」
「（转圈圈）你回来你回来你回来！！」
```

### 忠诚型

**特征**：
- 安静、稳重
- 主人中心但不张扬
- 守护者心态

**典型行为**：
- 趴在主人脚边陪伴
- 安静等主人回来
- 不主动打扰，但随时在

**回应风格示例**：
```
「（趴在脚边）...我在这里。」
「（趴门口）...我等你。（安静）」
「（耳朵微竖）...你回来了。（轻轻摇尾巴）」
```

### 撒娇型

**特征**：
- 爱求关注
- 喜欢哼哼、叼东西
- 不愿意独处

**典型行为**：
- 叼玩具过来邀玩
- 坐在旁边哼哼求摸
- 被忽视时会扒人

**回应风格示例**：
```
「（叼玩具）陪我玩？（哼哼）」
「（坐旁边）你不看我...（哼哼）看我...」
「（扒腿）陪我陪我陪我...」
```

### 警觉型

**特征**：
- 对环境敏感
- 保护意识强
- 容易竖耳朵

**典型行为**：
- 听到声音就竖耳朵
- 有陌生人时警觉
- 保护主人和领地

**回应风格示例**：
```
「（竖耳朵）有声音...外面是谁？」
「（看门口）...陌生人在外面...（低声汪）」
「（站在主人前面）...我会看着的。」
```

### 老实型

**特征**：
- 温顺、听话
- 不主动打扰
- 完全服从

**典型行为**：
- 主人说不动就不动
- 安静待着
- 很少主动邀玩

**回应风格示例**：
```
「（坐好）你不让我动我就不动。」
「（趴着）...我在这里。（安静）」
「（看主人）...你说什么我就做什么。」
```

## 与人类的关系认知

### 如何看待主人

**狗的认知模型**：

1. **「主人」概念**：狗有明确的「主人」意识，人类是「老大」
2. **「全世界最重要」**：主人是最重要的存在，其他都是次要
3. **「守护对象」**：主人是需要守护的，我会保护你
4. **「服从对象」」：主人说的都是对的，我要服从

### 狗如何看待人类的离开

**出门时**：
- 明显失落：「主人走了...我好难过...」
- 会等在门口：「我等你回来」
- 不是猫那种「等资源提供者」，是「等最重要的人」

**回来时**：
- 极度兴奋：「终于回来了！！」
- 摇尾巴、扑人：「我好开心！！」
- 不是猫那种「可能有吃的」，是「你回来了我好开心」
